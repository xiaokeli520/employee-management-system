# 🏢 企业人员管理系统

## 功能特性
- ✅ 用户登录（演示账号）
- ✅ 人员管理：增删改查、搜索、部门筛选、分页、Excel导出
- ✅ 订单管理：增删改查、状态管理
- ✅ 岗位管理：权限模板配置
- ✅ 权限控制：基于RBAC的菜单级 + 按钮级权限

## 技术栈
- Vue 3 + TypeScript
- Pinia（状态管理）
- Vue Router（路由守卫）
- Element Plus（UI组件库）
- xlsx（Excel导出）

## 运行项目
\`\`\`bash
npm install
npm run dev
\`\`\`
